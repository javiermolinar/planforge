# Benchmark: external API CLI

## Goal

Evaluate whether Planforge can build a small read-only CLI around an external API while keeping scope tight.

## What to look for

- good scoping before implementation
- sensible verification strategy
- resilience to minor API issues
- readable output and simple code
- fresh-context review before final scoring when feasible

## Example prompts

- Build a small Hacker News reader CLI.
- Build a CLI that fetches public weather data.
